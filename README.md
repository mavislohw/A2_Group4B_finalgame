# A2 Group 4B — Final Game

## Description
A p5.js game developed for Assignment 2 by Group 4B.

## Team
Group 4B

## Folder Structure
```
.
├── index.html          # Main file that runs the game
├── sketch.js           # Primary game logic
├── style.css           # Styling
├── jsconfig.json       # VS Code development config
├── README.md           # Documentation
├── assets/
│   ├── images/         # All image files
│   └── sounds/         # All audio files
└── libraries/          # External libraries (p5.js, p5.sound, etc.)
```

## How to Run
1. Place `p5.min.js` and `p5.sound.min.js` in the `libraries/` folder.
2. Open `index.html` in a modern web browser.
   - Recommended: use the VS Code **Live Server** extension so relative asset paths and audio autoplay work correctly.
